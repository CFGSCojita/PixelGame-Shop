Bienvenido a PixelGame Shop

PixelGame Shop es un proyecto integral de desarrollo web creado como trabajo académico que une tres asignaturas fundamentales: DIW (Diseño de Interfaces Web), DWEC (Desarrollo Web en Entorno Cliente) y DWES (Desarrollo Web en Entorno Servidor).

Este proyecto representa una experiencia completa de desarrollo de una tienda online de videojuegos, donde se aplican conceptos modernos de diseño responsive, interactividad con JavaScript, y gestión de bases de datos con PHP y MySQL.
